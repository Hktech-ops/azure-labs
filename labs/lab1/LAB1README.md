# Lab 1: RBAC Basics

## Objective
Create users, groups, and assign RBAC roles to demonstrate least privilege.

## Steps
1. Create a resource group
2. Create a user and group
3. Assign RBAC role
4. Test access
5. Remove access

## Scripts
- deploy.ps1: Creates RG and assigns role
- cleanup.ps1: Removes RG and role assignment
